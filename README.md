# react-cli

一个播种前端创建react模板的命令

## 使用方法

```bash
# 默认最新代码
npm install -g https://github.com/BozhongFE/react-cli
# tag 形式安装
npm install -g https://github.com/BozhongFE/react-cli/#v1.0.0
react-cli <command> [options]
```

## 命令列表

```bash
react-cli init [项目名称]
```

## 支持类型

+ Router
+ Router History Mode
+ Less
+ Redux
+ Redirected 地址重定向
+ Source 打包到 source 版本库

> Redirected启用时，不使用base标签，而是文件引用写死对应环境文件路径
